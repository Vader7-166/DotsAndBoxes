# DotsAndBoxes
1. Clone dự án về máy
2. Chạy lệnh python -m venv .venv để tạo môi trường ảo
3. Khởi chạy môi trường ào:
   - Windows: .venv\Scripts\activate
   - Linux / Mac: source .venv/bin/activate
4. Chạy lệnh pip install -r requirements.txt để cài các thư viện yêu cầu
5. Chạy python main.py để khởi chạy game
